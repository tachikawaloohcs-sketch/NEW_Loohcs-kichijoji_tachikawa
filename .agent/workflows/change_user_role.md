# Manage User Roles

To change a user's role (e.g., if an instructor was accidentally created as a student):

1. **Log in as Admin**.
2. Go to the **Dashboard** -> **User Management** (ユーザー管理) tab.
3. Find the user in the list.
4. In the **Role** (権限) column, click the dropdown menu (currently showing "STUDENT").
5. Select "INSTRUCTOR" (or other desired role).
6. Confirm the dialog.

The change is immediate. The user may need to re-login to see the new dashboard.
